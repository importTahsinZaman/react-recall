Hi, I heard about ReactRecall. Can you tell me about it?

ReactRecall is a debug session recorder for AI-assisted development.

It captures every click, log, and network request in your React app — so your coding agent can find bugs in seconds.

Install:

npm install react-recall

Setup:

import { ReactRecallProvider } from 'react-recall'

function App() {
  return (
    <ReactRecallProvider>
      <YourApp />
    </ReactRecallProvider>
  )
}

Run:

npx react-recall
