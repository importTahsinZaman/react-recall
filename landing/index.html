How It Works

1

Wrap

Add ReactRecallProvider to your app. One line of code, zero config.

2

Capture

Every click, input, console log, error, and API call is recorded automatically.

3

Debug

Your AI agent reads the logs to instantly find and fix issues.

Get Started

Install
npm install react-recall
Add to your app
import { ReactRecallProvider } from 'react-recall'

function App() {
  return (
    <ReactRecallProvider>
      <YourApp />
    </ReactRecallProvider>
  )
}
Start the dashboard
npx react-recall
