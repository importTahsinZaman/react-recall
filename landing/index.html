ReactRecall

Debug session recorder for AI-assisted development

Try the Demo

Click Send in the chat window and watch Claude Code debug the error using ReactRecall logs.

Install

npm install react-recall

Setup

import { ReactRecallProvider } from 'react-recall'

function App() {
  return (
    <ReactRecallProvider>
      <YourApp />
    </ReactRecallProvider>
  )
}

Run

npx react-recall
